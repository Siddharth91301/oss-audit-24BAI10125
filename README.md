## About the Project

This project is part of the Open Source Software coursework. It focuses on studying an open-source tool and understanding how it works in a real system environment.

For this audit, Python has been selected as the software of interest. The project explores its background, licensing, role in Linux systems, and how it fits into the larger open-source ecosystem.

Along with the theoretical study, a set of shell scripts has been created to demonstrate basic system-level operations.

---

## Selected Software

Python is a widely used programming language that is open source and supported by a large developer community. It is known for its readability and is commonly used in areas such as automation, data analysis, and web development.
